# MineCart

Ванильный Minecraft сервер с дружелюбным сообществом и минимальным вмешательством в геймплей.

## О сервере

- Ванильный геймплей без pay-to-win механик
- Голосовой чат
- Кастомные крафты и расширенные команды
- Защита территорий и построек
- Профессиональная модерация
- Различные самописные плагины
